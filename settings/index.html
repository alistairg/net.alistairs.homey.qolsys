Qolsys IQ Panel — Backup & Restore

Export pairing data

Pairing with the panel is a one-time exchange that consumes one of the panel's IQ Remote slots (3 on IQ Panel 2/2+, 8 on IQ Panel 4 / IQ Pro). Freeing a slot generally requires alarm-company intervention. Export now, store the file somewhere safe (1Password / encrypted note / similar), and you can restore the pairing instantly if Homey is wiped or the app is reinstalled.

The export contains the RSA private key that authenticates Homey to your panel as a trusted IQ Remote. Treat the file like a credential: don't email it, don't paste it in chat, don't commit it to git.

Restore pairing data

Upload a backup file previously exported from this app. The file replaces the current PKI material and panel IP in app settings, and reconnects to the panel using the restored credentials.
