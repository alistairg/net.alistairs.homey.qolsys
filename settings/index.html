Qolsys IQ Panel — Backup & Restore

Export pairing data

Pairing with the panel consumes one of the panel's IQ Remote slots (3 on IQ Panel 2/2+, 8 on IQ Panel 4 / IQ Pro). Freeing a slot generally requires alarm-company intervention. Export now, store the result somewhere safe (e.g. 1Password / encrypted note), and you can restore the pairing instantly if Homey is wiped or the app is reinstalled.

The export contains the RSA private key that authenticates Homey to your panel as a trusted IQ Remote. Treat it like a credential.
